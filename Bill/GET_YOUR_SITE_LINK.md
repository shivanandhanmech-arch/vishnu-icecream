# How to Get Your Site Link

## üîç Local Server (For Testing on Your Computer)

### If server is running:
Your site link is: **http://localhost:8000**

### To start server:
1. Double-click `start-server.bat`
2. Keep the window open
3. Open browser: http://localhost:8000

### To check if server is running:
- Double-click `check-server.bat`
- It will tell you if server is running and open the site

## üåê Deployed Site (For Sharing with Friends)

### If you haven't deployed yet:

**Option 1: GitHub Pages (Recommended - Permanent & Free)**
1. Go to: https://github.com
2. Create account and repository
3. Upload your files
4. Enable GitHub Pages
5. Get link: `https://your-username.github.io/vishnu-icecream`

**Option 2: Vercel (Quick & Free)**
1. Go to: https://vercel.com
2. Sign up (free)
3. Drag and drop your `Bill` folder
4. Get instant link: `https://your-project.vercel.app`

**Option 3: Netlify (But use account, not Drop)**
1. Go to: https://app.netlify.com
2. Sign up (free)
3. Deploy manually
4. Get link: `https://your-site.netlify.app`

## üöÄ Quick Actions

### See your local site:
- Double-click `check-server.bat` (checks and opens if running)
- Or double-click `start-server.bat` (starts server)
- Then visit: http://localhost:8000

### Deploy permanently:
- Double-click `DEPLOY_GITHUB_PAGES.bat` (guides you through GitHub Pages)
- Or double-click `DEPLOY_VERCEL.bat` (guides you through Vercel)
