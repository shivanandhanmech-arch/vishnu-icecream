# Deploy Your Website Permanently - Quick Guide

## Option 1: Netlify Drop (Easiest - 2 Minutes!)

### Steps:
1. **Keep your files ready**: Make sure you have these 3 files in your `Bill` folder:
   - `index.html`
   - `styles.css`
   - `script.js`

2. **Go to Netlify Drop**:
   - Visit: https://app.netlify.com/drop
   - No account needed for basic use!

3. **Drag and Drop**:
   - Simply drag your entire `Bill` folder (or the 3 files) onto the Netlify Drop page
   - Wait a few seconds...

4. **Get Your Permanent URL**:
   - You'll get a URL like: `https://random-name-123.netlify.app`
   - This URL works permanently and on mobile!
   - Share this URL with your friend

5. **Done!** Your website is now live permanently!

---

## Option 2: GitHub Pages (Free Forever)

### Steps:
1. Create a GitHub account (if you don't have one): https://github.com/signup

2. Create a new repository:
   - Click "New" on GitHub
   - Name it: `vishnu-icecream` (or any name)
   - Make it Public
   - Don't initialize with README
   - Click "Create repository"

3. Upload your files:
   - Click "uploading an existing file"
   - Drag and drop: `index.html`, `styles.css`, `script.js`
   - Click "Commit changes"

4. Enable GitHub Pages:
   - Go to Settings â†’ Pages
   - Under "Source", select "main" branch
   - Click "Save"
   - Wait 1-2 minutes

5. Get your URL:
   - Your site will be at: `https://your-username.github.io/vishnu-icecream`
   - Share this URL!

## Option 3: Vercel (Also Very Easy)

1. Visit: https://vercel.com
2. Sign up (free)
3. Click "Add New Project"
4. Drag and drop your folder
5. Get instant URL!
