# How to Find Your GitHub Pages Link

## üìç Where to Find Your Link

### Step 1: Go to Your Repository
1. Open your GitHub repository (the one you created)
2. Make sure you're on the main page

### Step 2: Check Settings > Pages
1. Click **"Settings"** tab (top right of your repository)
2. Scroll down to **"Pages"** section (left sidebar)
3. Your link will be shown at the top of the Pages section

### Step 3: Your Link Format
Your link will be:
```
https://YOUR-USERNAME.github.io/REPOSITORY-NAME
```

**Example:**
- If your username is `john123`
- And repository name is `vishnu-icecream`
- Your link will be: `https://john123.github.io/vishnu-icecream`

### If Link Doesn't Show Yet:
1. **Wait 1-2 minutes** - GitHub needs time to build your site
2. **Refresh the Settings page** - Press F5
3. **Check the Pages section** - Look for green checkmark or "Your site is live at..."

### If You See "Your site is ready to be published":
- Make sure you selected **"main"** branch
- Make sure you selected **"/ (root)"** folder
- Click **Save** again
